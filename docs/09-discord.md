# Discord Integration Guide

The Discord integration allows creators to receive automated notification updates, real-time draft notifications, and external review links directly inside their Discord server.

## Overview
PostSync leverages Discord **incoming webhooks** and the Discord API to dispatch formatted rich embed cards to designated channels.

---

## 1. Setup Instructions

To connect Discord to your PostSync workspace:

1.  **Generate a Webhook URL on Discord:**
    *   Open Discord and navigate to the channel where you want to receive alerts (e.g., `#content-alerts`).
    *   Click the **Gear Icon (Channel Settings)**.
    *   Select **Integrations** from the sidebar, then click **Webhooks**.
    *   Click **New Webhook**, name it (e.g., `PostSync Alerts`), and select the desired channel.
    *   Click **Copy Webhook URL**.

2.  **Bind Webhook in PostSync Dashboard:**
    *   Go to **Settings > Integrations** or **Automation Settings**.
    *   Find the **Discord webhook** field.
    *   Paste the copied Discord webhook URL and click **Save**.
    *   Click **Send Test Message** to verify the handshake is successful.

---

## 2. Supported Features

*   **Real-time Notifications:** Instantly get notified when posts are successfully scheduled, synced, or published.
*   **Draft Review System:** Receive notifications for new drafts generated by the PostSync Automation Engine.
*   **External Approval Handshake:** Embeds contain tokenized links allowing workspace members to **approve** or **reject** drafts directly from Discord without logging into the dashboard.

---

## 3. Webhook Embed Format
Alert messages are sent as structured embeds:
*   **Embed Color:** `#5865F2` (Discord Blurple)
*   **Author Line:** PostSync Notification Engine
*   **Fields:**
    *   `Topic`: Trending topic title or draft slug.
    *   `Draft Caption`: The adapted post copy.
    *   `Platforms`: Destination channels (e.g., LinkedIn, Instagram).
*   **Interactive Review Actions:** DMs/channels will display markdown links in the format:
    `[Approve Draft](https://<domain>/api/automation/external-approve?token=<jwt>&action=approve)`
    `[Reject Draft](https://<domain>/api/automation/external-approve?token=<jwt>&action=reject)`
